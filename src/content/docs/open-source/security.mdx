Do not open a public GitHub issue for suspected security vulnerabilities.

## Report Privately

Email `luca@refactoring.club` with the subject line `[Tolaria Security]`.

Include as much as you can:

- A short description.
- Reproduction steps or proof of concept.
- Affected version or commit.
- Impact assessment.
- Suggested mitigation if you have one.

## Supported Scope

The current policy supports the latest stable release and best-effort fixes on
`main`. Older releases and prereleases may not receive fixes.

## Disclosure

Give the maintainers reasonable time to investigate and ship a fix before
publishing details.
