Tolaria is a desktop app for Markdown knowledge bases. It runs locally and does
not require an account, subscription, or hosted Tolaria service.

## Download

Download the latest release from the official download page:

[Download Tolaria](https://refactoringhq.github.io/tolaria/download/)

Tolaria is designed for local-first use. Your vault is a folder of Markdown files,
and Git support uses your existing Git setup and remotes.

## Requirements

- A desktop operating system supported by the current release.
- Git installed if you want commit, history, push, pull, and AutoGit features.
- A signed-in CLI agent such as Claude Code, Codex CLI, or Gemini CLI if you want
  to use external AI tooling with your vault.

## First Launch

On first launch, choose one of these paths:

- **Clone the Getting Started vault** if you want a guided walkthrough with sample notes.
- **Open an existing folder** if you already keep Markdown notes.
- **Create or clone another Git-backed vault** if you want to start from your own remote.

The fastest path is the Getting Started vault because it teaches the sidebar,
note list, editor, properties panel, bottom bar, Git workflow, and AI setup in
one small workspace.
