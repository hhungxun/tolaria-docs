## Open

Use `Cmd+K` on macOS or the equivalent control shortcut on other platforms.

## Common Uses

- Create a note.
- Navigate to a section, type, or note.
- Open raw mode.
- Reload the vault.
- Commit or sync.
- Set up external AI tools.
- Run quick AI prompts.

## Quick Prompt Mode

The command palette can become a quick AI prompt surface. Add wikilinks when the
assistant should receive specific note context.

## Matching

The palette supports flexible matching, so you do not need to remember the exact
command wording.
