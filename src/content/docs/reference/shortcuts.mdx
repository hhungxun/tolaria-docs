Tolaria is keyboard-first. Exact shortcuts can evolve, so use the command palette
and app menus as the source of truth in your installed version.

## Common Shortcuts

| Action | Shortcut |
| --- | --- |
| Open command palette | `Cmd+K` |
| Create a note | `Cmd+N` |
| Toggle raw mode | `Cmd+/` |
| Toggle favorite | `Cmd+D` |
| Mark organized | `Cmd+E` |
| Open settings | `Cmd+,` |
| Navigate back or forward | `Cmd+[` / `Cmd+]` |

## Discover More

Open the command palette and type what you want to do. Tolaria's command matching
is designed for discovery rather than memorization.

## macOS Note

Some Option-key combinations produce special characters on macOS. Tolaria's
shortcuts are designed around command-based accelerators where possible.
