The note list shows the notes selected by the current sidebar item, search, view,
folder, or relationship neighborhood.

## Sorting

Sort by title, modified time, created time, or note properties. Some sort choices
are remembered for the current section or type.

## Display Properties

The note list can show selected frontmatter properties as columns. This is useful
for project status, dates, owners, URLs, or relationships.

## Search Within a List

List search narrows the current set of notes. Use it when you already know the
right type, view, or folder and need a smaller result set.

## Neighborhood Mode

Neighborhood mode pivots the note list around one note. The source note stays
pinned, outgoing relationships appear first, inverse relationships follow, and
related notes can appear in more than one group if multiple relationships apply.

Use Neighborhood mode when you want to move through connected Projects, People,
Topics, or source notes without returning to the sidebar.
