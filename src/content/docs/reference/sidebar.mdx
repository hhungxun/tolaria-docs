The sidebar is the first navigation layer in Tolaria.

## Sections

- **Inbox**: notes that have not been marked organized.
- **All Notes**: every note in the active vault.
- **Archive**: notes kept out of regular navigation.
- **Favorites**: pinned notes that should stay close.
- **Views**: saved YAML filters.
- **Types**: note categories such as Project, Topic, Person, and Note.
- **Folders**: filesystem folders inside the vault.

## Type Navigation

Clicking a type shows all notes with matching `type:` frontmatter. If the type
has a type document in `type/`, Tolaria can use its icon, color, order, sort, and
list display settings.

## Folder Navigation

Folders are available, but Tolaria treats them as secondary structure. Use them
for filesystem organization, attachments, or specialized project areas.

## Drag and Drop

Dropping a note on a type changes its `type:` frontmatter. Dropping a note on a
folder moves the file. Both operations keep the vault as ordinary files on disk.
