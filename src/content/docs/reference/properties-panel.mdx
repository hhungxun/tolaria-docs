## Common Properties

- `type`: the note type.
- `status`: lifecycle state.
- `url`: external source link.
- `date`, `start_date`, `end_date`: date metadata.
- `belongs_to`, `related_to`, `has`: relationship fields.
- `icon`: per-note icon.

## Relationship Editing

Any frontmatter field containing wikilinks becomes a relationship. The Properties
panel presents those values as navigable links.

## Type-Aware Controls

Tolaria can render different controls for text, numbers, dates, status fields,
tags, URLs, icons, and wikilink relationships.

## System Properties

Fields beginning with `_` are Tolaria-managed system properties. They are hidden
from normal property surfaces but remain visible in raw Markdown.
