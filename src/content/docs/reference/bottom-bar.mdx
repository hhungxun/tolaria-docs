The bottom bar keeps operational state visible without becoming the main workspace.

## Vault Selector

Switch active vaults, open another local folder, or clone a vault from a Git URL.

## Git Status

Git-backed vaults show pending changes, commit access, push and pull state,
history, and remote connection state.

## Pulse

Pulse is a lightweight Git activity view. Use it to inspect recent vault changes
by time.

## AI Status

The AI status area shows available local agents and the selected agent. Use it to
confirm whether tools such as Claude Code, Codex CLI, or Gemini CLI are available
to Tolaria.

## Feedback and Version

The bottom bar also exposes app version information and feedback entry points.
