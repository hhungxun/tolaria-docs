The Inbox is a temporary state for notes that are not organized yet.

## Capture First

When you are working, do not interrupt yourself to place every note perfectly.
Create the note, paste the link, add the thought, and move on.

## Review on a Cadence

At least weekly, open the Inbox and process each note:

1. Delete notes that are no longer useful.
2. Archive notes that should be retained but not acted on.
3. Keep notes that should connect to a current system.
4. Add type, status, and relationships where they help.
5. Mark the note organized.

## What Organized Means

An organized note has enough context for future you or an AI agent to understand
why it exists. It might belong to a project, responsibility, topic, person,
procedure, or evergreen idea.

## Keep the Inbox Honest

Do not use the Inbox as a permanent backlog. If a note keeps surviving review but
never gets connected, either archive it or create the missing project or topic
that gives it meaning.
